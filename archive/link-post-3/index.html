Архив клуба по разработке интерфейсов

Подборка ссылок, выпуск третий

Опубликовано Thu Jul 24 2014 14:22:34 GMT+0400 (MSK).

Вёрстка

  • Unquoted font family names in CSS

    Mathias Bynens написал хорошую заметку про то, когда нужны кавычки у значений свойства font-family. Автор подробно рассматривает различные варианты и, ссылаясь на спецификации, объясняет почему, например, sans-serif и 'sans-serif' — не одно и то же.

  • Animated mask icons

    Simurai сделал небольшую демонстрацию использования CSS-масок — проприетарного для WebKit свойства, которое пока не описано в спецификациях. Хотя практической ценности этот эксперимент не имеет, это хороший взгяд в будущее — с использованием масок можно делать довольно много интересных эффектов, в частности, если посмотреть этот эксперимент в последних сборках WebKit, то можно увидеть, как к ним применяются CSS-фильтры.

  • Media Query & Asset Downloading Results

    Tim Kadlec провёл интересное исследование того, как нужно скрывать изображения, чтобы браузеры не посылали к ним запросы. Это особенно актуально для адаптивных раскладок — когда может возникнуть желание показывать разные изображения для разных устройств.

  • Alternative Units for CSS3 Rotation Transforms

    Louis Lazaris написал небольшую заметку, описывающую альтернативные единицы измерения углов для CSS-трансформаций.

  • Cross browser drop caps with CSS

    Ian Devlin поделился рецептом создания эффекта буквицы при помощи псевдоэлемента :first-letter. У этого псевдоэлемента до сих пор есть множество особенностей отображения в разных браузерах, но автор показывает, как можно их обойти и достичь почти одинакового отображения буквицы во всех браузерах.

Дизайн

  • Mobile Navigation Design & Tutorial

    В этой статье Nick La описывает различные варианты того, как можно изменять навигацию на страницах в рамках адаптивной раскладки. Автор перечисляет несколько стандартных способов улучшения навигации для мобильных устройств, а затем подробно расписывает один из вариантов.

  • Responsive Layouts, Responsively Wireframed

    James Mellers сделал страницу с демонстрацией того, как могут работать адаптивные раскладки. Кроме этого, автор ставит проблему отсутствия инструмента прототипирования адаптивных раскладок.

Остальное

  • zxcvbn: realistic password strength estimation

    В техническом блоге компании Dropbox можно прочитать довольно подробное описание алгоритма проверки паролей на сложность. Для этого в компании создали открытый инструмент, который можно найти на Гитхабе. Инструмент в действии можно посмотреть на демонстрационной странице — там можно проверить различные пароли на сложность и увидеть подробный разбор введённого пароля с точки зрения алгоритма.

  • Licensing for JavaScript Projects

    Alex R Young написал немного про лицензирование JavaScript-проектов: как и зачем нужно предоставлять лицензию для своего открытого проекта.

  • New tricks in XMLHttpRequest2

    Eric Bidelman подробно расписал новшества новой спецификации XHR. Автор рассказывает про получение и отправку данных, про CORS и приводит несколько практических примеров.
