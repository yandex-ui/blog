Подборка ссылок, выпуск четвёртый

Опубликовано Thu Jul 24 2014 14:22:45 GMT+0400 (MSK).

Вёрстка

  • Let’s Talk about Semantics

    Mike Robinson написал статью про семантику в контексте HTML5. Помимо прочего, в статье есть ссылка на схему HTML5 Element Flowchart. Эта схема поможет определиться что использовать при вёрстке очередного семантичного сайта — section, aside, div или что-то иное.

  • Cicadients

    Eric Meyer реализовал на CSS-градиентах пару примеров «принципа цикады», который описал Alex Walker в прошлом году. Если вы не знакомы с этим принципом, стоит перечитать оригинальную статью или перевод.

  • Last week in WebKit: CSS’ -khtml- and -apple-, and ancestorOrigins

    Peter Beverloo описал изменения в движке WebKit за последнюю неделю. В число различных исправлений вошло, например, следующее: теперь при отстуствии заданного цвета для свойства box-shadow будет использоваться значение color для этого элемента. Такое поведение диктуется спецификациями и уже работает в FireFox. Кроме того, из WebKit начали убрирать поддержку префиксов -khtml- и -apple-.

  • Responsive viewport units

    David Storey написал заметку про новые единицы измерения в CSS — vh, vw, и vmin.

JavaScript

  • Beercamp: An Experiment With CSS 3D

    Tom Giannattasio в статье для Smashing Magazine написал о том, как создавался сайт для конференции Beercamp 2012. Этот сайт — эксперимент на тему 3D в CSS. Всем интресующимся этой темой рекомендуется изучить как саму статью, так и сам сайт. В статье описываются проблемы, возникшие при создании сайта, и приводится несколько примеров работы с 3D.

  • Silly list scroll effects

    Hakim El Hattab проэкспериментировал с анимацией скроллящихся блоков. Хотя практическое применение такой технике сложно найти, эксперимент стоит изучить изнутри — там применяется несколько интересных решений, например, использование requestAnimationFrame для более плавной анимации.

  • Of parser-fetishists and semi-colons

    Christian Heilmann написал статью по мотивам нашумевшей драмы о символе ;. В статье автор рассуждает о том, надо ли писать код только для браузеров или следует в первую очередь ориентироваться на разработчиков. Также затронуты проблемы использования гибкого синтаксиса языков.

  • JavaScript API Яндекс.Карт — версия 2.0

    Яндекс.Карты выпустили новую версию API, в статье для Хабрахабра описываются задачи, которые стояли перед разработчиками при создании API.

Адаптивные раскладки

  • Responsive web design: a project-management perspective

    Rudy Rigot написал статью о том, чем нужно руководствоваться и о чём думать при начале работы над проектом, если в нём планируется использовать адаптивную раскладку. Автор задаётся такими вопросами как «а стоит ли вообще использовать адаптивную раскладку?», «нужно ли начинать думать о раскладке с мобильных устройств?» и многими другими.

  • Multi-Device Layout Patterns

    Luke Wroblewski описал существующие варианты адаптивных раскладок. К каждому варианту автор приводит несколько ссылок на сайты, использующие соответствующие раскладки.

  • Creating a mobile-first responsive web design

    Brad Frost написал подробную статью о подходе «mobile-first» при создании сайтов с адаптивной раскладкой. Автор пошагово демонстрирует создание примера для статьи, расписывая как вёрстку, так и использование различных скриптов для достижения оптимального эффекта от выбранного подхода.

Остальное

  • Augmented Paper

    Matt Gemmell написал интересную статью про современные интерфейсы и концепцию, которую автор назвал «Augmented Paper».

  • A Baseline for Front-End Developers

    Rebecca Murphey рассуждает о том, как изменилась клиентская разработка за последние несколько лет. В посте перечисляется большое число инструментов и библиотек, с которыми имеет смысл ознакомиться разработчикам.
