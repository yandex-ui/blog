Архив клуба разработки интерфейсов

Подборка ссылок, выпуск двенадцатый

Опубликовано Thu Jul 24 2014 14:23:54 GMT+0400 (MSK).

Вёрстка

  • Principles of writing consistent, idiomatic CSS, перевод на русский

    Nicolas Gallagher описал свои принципы того, как нужно писать CSS. Автор не приводит какого-то единого решения, но даёт, по сути, пошаговое руководство по созданию своего стайлгайда для CSS-кода.

  • The CNS Color Naming System

    Tab Atkins предлагает добавить возможность в CSS описывать цвета более интуитивно — по системе CNS. Эта система предполагает, что цвет можно задавать c помощью ключевых слов вроде vivid purplish blue, при этом считается, что точный оттенок не так важен, как простота определения цвета. Это пока только идея, и не факт, что дело дойдёт до спецификации, но, например, для какого-либо CSS-препроцессора такое точно можно было бы реализовать.

  • 3D thumbnail hover effects

    В блоге Codrops появилось очередное интересное демо от Mary Lou — ховер с 3D эффектом для превью изображений.

  • Perfectly Rotate and Mask Thumbnails With CSS3

    Jim Nielsen также написал туториал по созданию одного из вариантов превью для изображений. В статье описывается применение CSS трансформации вращения и следующие из него нюансы.

  • Single-direction margin declarations

    Harry Roberts советует всегда использовать только одно направление для маджинов — например, margin-bottom для вертикальных отступов и margin-left для горизонтальных. В его статье объясняется, почему он пришёл к такому правилу.

  • Foundation 3.0 Typography and Modular Scale Collide

    Chris Michel описал подход к выбору размера шрифта для разных уровней заголовков, в основе которого лежит автоматическая генерация всех размеров с использованием Sass.

  • An Introduction to WebVTT and &lt;track&gt;

    Ian Devlin написал статью про использование тега <track>, появившегося в HTML5.

  • Why Aren’t You Using SVG?

    Jonathan Cutrell в своей статье приводит доводы в пользу SVG и рассказывает о том, почему эту технологию можно использовать уже сейчас.

  • CSS generated content on replaced elements

    В нескольких последних демонстрациях возможностей CSS часто используются псевдоэлементы для замещаемых элементов (img, input и т.п.) Catalin Rosu спросил у разных разработчиков что они думают на этот счёт, и привёл их ответы.

JavaScript

  • CSS 3D lighting engine

    Tom Giannattasio, известный своими экспериментами над трёхмерным CSS, представил своё новое творение — JS-библиотеку, рассчитывающую освещение для трёхмерных объектов в CSS. Этот эксперимент пока работает только в браузерах на движке WebKit, но на него стоит посмотреть, чтобы понимать, что можно будет создавать в ближайшем будущем.

  • YSlow development: getting started, setup, custom rulesets

    Stoyan Stefanov начал публикацию серии статей про использование инструмента YSlow.

Дизайн

  • A Responsive Design Case Study

    David Bushell описал, как он создавал очередной сайт на адаптивной раскладке и какие решения он принимал по различным вопросам, возникающих при этом.

  • Fluid Type

    Одним из важных моментов при создании адаптивных раскладок является типографика. Trent Walton пишет о том, какой подход он выбрал для адаптивной типографики.

Другое

  • Mobile Considerations in User Experience Design: “Web or Native?”

    В Smashing Magazine опубликовали одну из частей вышедшей книги Smashing Book 3. В ней Aral Balkan пишет о том, что использовать при разработке интерфейса под мобильные устройства — нативные средства или же веб-приложение.

  • Shiny Demos

    Разработчики из Оперы запустили сайт, на котором собрали разные демо современных технологий. Там можно посмотреть и послушать HTML5 аудио, видео, почитать код CSS3-примеров и многое другое. Кроме того, код всех примеров доступен и на Гитхабе.

  • CodePen

    Alex Vazquez, Tim Sabat и Chris Coyier запустили новый проект — очередную песочницу для примеров, в которых можно использовать не только HTML/CSS/JS, но и, например, Haml/Sass/CoffeeScript.
